Omega Sidebar Navigation

Omega est une jolie sidebar responsive en pure javascript

Télécharger Omega SidebarCe site de démonstration utilise le framework CSS Milligram

Démonstration ?

Pour une démonstration, diminuez la fenêtre de votre navigateur ou visitez simplement ce site internet sur votre tablette/smartphone et cliquez sur l'icône ☰

Utilisation

1.Omega Sidebar est très simple d'utilisation. Commencez par charger les fichiers CSS et Javascript.

CSS : 

<link rel="stylesheet" href="path/to/omega.css">
JS : 
<script src="path/to/omega.js"></script>

2.Copiez ensuite le code HTML du menu et de la sidebar. (bien entendu, à modifier selon votre projet)

HTML : 


<div id="omega">
<a href="#" id="logo">Sitename</a>
<div id="omega-content">

<nav>
<ul>
<li><a href="#">Home</a></li>
<li><a href="#">Page 1</a></li>
<li><a href="#">Page 2</a></li>
<li><a href="#">Contact</a></li>
</ul>
</nav>

<div class="omega-links">
<a href="#" class="button">Inscription</a>
<a href="#" class="button button-outline ">Se connecter</a>
</div>
</div>

<button id="omega-button">&#9776;</button>
<div id="omega-sidebar">
<div id="omega-sidebar-header"></div>
<div id="omega-sidebar-body"></div>
</div>
<div id="omega-overlay"></div>

</div>

3.L'animation de la sidebar se trouve dans l'ID #omega-sidebar du fichier omega.css. Vous pouvez la modifier comme bon vous semble. 


#omega-sidebar{
...
transition: transform .2s ease-in-out;
}

4.Les icônes du menu de la sidebar utlisent la librairie FontAwasome avec leurs Unicodes associés

Il est important d'ajouter un \ avant chaque Unicode


#omega-sidebar-body nav ul li:first-child:before {
    content: "\f015";
    font-family: "Font Awesome 5 Free";
    font-weight: 900;
}
Chaque icône utilise le sélecteur :nth-child suivant l'ordre d'apparition des liens du menu. 

#omega-sidebar-body nav ul li:nth-child(2):before {
    content: "\f15c";
}
